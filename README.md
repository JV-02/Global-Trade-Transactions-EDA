# Global-Trade-Transactions-EDA
Exploratory data analysis and visualization of global trade transaction data using Python.

## Objective
Perform exploratory analysis on trade transaction data to uncover country-wise and category-wise trade trends.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn

## Workflow
- Data cleaning
- Aggregations using groupby
- Descriptive statistics
- Visualization of trends
- Insight generation

## Key Insights
- Trade volume distribution across countries
- Category-wise patterns
- Correlation analysis

## Files
- trade_analysis.ipynb

## How to Run
Run the notebook to reproduce analysis and visualizations.
